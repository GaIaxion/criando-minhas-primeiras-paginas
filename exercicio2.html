HTML

            HTML é uma linguagem de marcação utilizada para desenvolver sites para a web. 
        É composto por elemenrtos definidos por meio de tags e suporta estilização 
        e script como por exemplo o JavaScript

CSS

        O CSS é a sigla para Cascade stylae e sheelt ou folha de estilos em cascata.
        É utilizado para estilizar componentes de uma página HTML.
        Pode ser utilizado online dentro da tag , ou fora dela em uma tag style ou até mesmso em um arquivo externo.
